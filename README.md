# Linear-Regression-Prediction
使用多元线性回归预测北京房价。Use multiple linear regression to forecast Beijing house price.  
输出线性回归模型分别在训练集和测试集上的效果图，以及在训练集和测试集上的误差。 
最后可以交互式得进行房价预测。
